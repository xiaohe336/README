# ROS2 摄像头标定使用指南

本文档记录了在 Jetson 板子上对单目摄像头（usb_cam）进行棋盘格标定的完整流程，以及生成的相机内参（Camera Matrix）和畸变系数（Distortion）。

## 1. 工作空间与路径
- **标定工作路径**：`/home/admin1/catkin_ws/src/apriltag_detection`
- **ROS2 版本**：Humble

## 2. 标定操作步骤

请打开三个不同的终端窗口，依次执行以下命令：

**终端 1：启动摄像头节点**
```bash
source /opt/ros/humble/setup.bash
ros2 run usb_cam usb_cam_node_exe
```

**终端 2：启动标定程序**
*注：使用的棋盘格内部角点为 11x8，单个方格边长为 0.0165 米 (1.65 cm)。*
```bash
source /opt/ros/humble/setup.bash
ros2 run camera_calibration cameracalibrator \
--size 11x8 \
--square 0.0165 \
--ros-args \
-r image:=/image_raw \
-p camera:=/usb_cam
```
*(在标定窗口中不断移动棋盘格，直到 `CALIBRATE` 按钮变绿，点击计算并 `SAVE`)*

**终端 3：查看实时画面（可选，用于调试）**
```bash
source /opt/ros/humble/setup.bash
ros2 run rqt_image_view rqt_image_view
```

## 3. 标定结果参数 (复制到代码中使用)

点击标定界面的 `SAVE` 后，完整数据会打包保存在 `/tmp/calibrationdata.tar.gz`。为了方便后续开发（如 ArUco 姿态估计），这里提取了核心的**内参矩阵 (K)** 和**畸变系数 (D)**。

根据你的摄像头分辨率需求，在代码中选择对应的一组：

### 选项 A：标准分辨率 (640 x 480)
适用于对帧率要求较高、运算资源有限的场景（如 `aruco_cam6.py` 默认配置）。
```python
# 480p 相机内参矩阵
camera_matrix = np.array([
    [437.159542, 0.0,       314.694745],
    [0.0,       437.261460, 224.603979],
    [0.0,       0.0,       1.0]
], dtype=np.float32)

# 480p 畸变系数
dist_coeffs = np.array([
    0.080439, -0.066403, 0.000416, -0.000629, 0.0
], dtype=np.float32)
```

### 选项 B：高清分辨率 (1920 x 1080)
适用于对检测精度和远距离识别要求更高的场景。
```python
# 1080p 相机内参矩阵
camera_matrix = np.array([
    [1091.869767, 0.0,       943.246982],
    [0.0,         1090.142844, 500.893113],
    [0.0,         0.0,       1.0]
], dtype=np.float32)

# 1080p 畸变系数
dist_coeffs = np.array([
    0.077628, -0.066885, -0.000425, -0.000008, 0.0
], dtype=np.float32)
```
